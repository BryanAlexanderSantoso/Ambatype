# Ambatype

A high-performance minimalist typing test application mimicking the Claude AI design aesthetic.

## Features
- Minimalist Claude AI Light & Dark modes.
- Real-time Words Per Minute (WPM) & Accuracy tracking.
- Test modes: Time (15, 30, 60, 120s), Words, Quote, Zen.
- Searchable language dataset.
- Post-test Results Dashboard with Data visualization.
- Responsive design.

## Stack
React 19, TypeScript, Vite, Tailwind v4, Lucide React, Framer Motion, Recharts.
